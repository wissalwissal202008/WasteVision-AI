Objectif de l’app

Aider les utilisateurs à trier les déchets facilement grâce à la reconnaissance d’objets par IA.

Educatif et ludique : l’utilisateur apprend en scannant.

Motivant : suivi des statistiques et badges.

2️⃣ Style général

Couleurs principales :

Vert 🌿 → recyclable

Bleu 💧 → verre

Jaune ⚡ → plastique

Gris 🗑 → non-recyclable

Typographie : simple, lisible, friendly

Ambiance : moderne, épurée, joyeuse, écologique

Icônes : claires et intuitives (caméra, recyclage, correction, tableau de bord)

3️⃣ Écrans principaux et éléments
3.1 Home / Scan Screen

But : scanner ou importer une image.

Éléments :

Bouton “Scanner un objet” → ouvre la caméra

Bouton “Importer depuis la galerie”

Flux caméra (optionnel)

Menu / barre de navigation vers Dashboard

3.2 Result Screen

But : afficher les résultats IA.

Éléments :

Nom de l’objet détecté

Matériau

Catégorie de déchet (avec couleur correspondante)

Conseils de recyclage et impact environnemental

Bouton “Corriger l’IA”

3.3 Correction Screen

But : corriger un objet mal identifié.

Éléments :

Champ pour modifier le nom de l’objet

Liste déroulante pour choisir la catégorie correcte

Bouton “Envoyer correction”

3.4 Dashboard / Statistics Screen

But : suivre impact écologique et motivation.

Éléments :

Nombre d’objets triés

Graphiques simples : répartition par matériau

Badges / récompenses

Design motivant et coloré

4️⃣ Flux utilisateur

L’utilisateur ouvre l’app → Home / Scan

Scanne un objet → Result Screen

Corrige si nécessaire → Correction Screen

Accède au Dashboard pour suivre ses statistiques

5️⃣ Boutons et interactions
Bouton / Action	Fonction	Style conseillé
Scanner un objet	Ouvre la caméra	Bouton rond coloré, icône caméra
Importer depuis la galerie	Importer une photo existante	Bouton rectangulaire, icône image
Corriger l’IA	Ouvre formulaire correction	Bouton secondaire, couleur contrastante
Envoyer correction	Envoie correction au backend	Bouton vert lumineux, feedback immédiat
Accéder au Dashboard	Aller aux statistiques	Icône tableau de bord, visible en haut
6️⃣ Notes pour Figma / Designer

Prévoir UI responsive pour mobiles Android et iOS

Icônes simples, lisibles, flat design

Utiliser feedback visuel pour chaque action (ex : confirmation pour correction envoyée)

Couleurs cohérentes avec le type de déchet pour faciliter la lecture

Ajouter animations légères pour rendre le scan et les badges motivants

💡 Optionnel :

Mode nuit / clair

Micro animations pour rendre le tableau de bord plus interactif

Gamification : étoiles ou badges pour chaque objet bien trié
