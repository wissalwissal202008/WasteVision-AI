🖥️ WasteVision AI – Frontend Description
Technologie

Framework : Flutter (cross-platform Android/iOS)

Langage : Dart

Design : Material Design moderne, couleurs vertes et naturelles pour rappeler l’écologie 🌱♻️

Compatibilité : mobile (smartphone), responsive, UI simple et intuitive

Écrans principaux
1️⃣ Accueil / Scan

Fonction : Permet à l’utilisateur de prendre une photo ou importer une image depuis la galerie.

Éléments :

Bouton “Prendre une photo”

Bouton “Importer depuis la galerie”

Mini-guide rapide pour l’utilisateur (“Prenez une photo d’un objet pour savoir où le recycler”)

2️⃣ Résultat / Analyse

Fonction : Affiche les résultats après analyse par l’IA.

Éléments :

Nom de l’objet détecté

Matériau (plastique, verre, métal, papier, etc.)

Catégorie de déchet (Recyclable, Compost, Non recyclable…)

Conseil écologique (ex : “Placez dans la poubelle jaune”)

Bouton de correction si l’IA se trompe

3️⃣ Correction utilisateur

Fonction : Permet à l’utilisateur de corriger l’objet mal identifié.

Éléments :

Champ pour entrer le nom correct de l’objet

Liste déroulante pour sélectionner la catégorie correcte

Bouton “Envoyer correction”

Système pour apprendre de la correction et améliorer l’IA

4️⃣ Tableau de bord écologique

Fonction : Affiche les statistiques personnelles de tri et impact environnemental.

Éléments :

Nombre d’objets triés

Types de déchets triés

Impact écologique estimé (ex : kg de CO₂ économisés)

Badges ou récompenses pour motiver l’utilisateur

Fonctionnalités supplémentaires

Navigation simple avec Bottom Navigation Bar ou Drawer menu

Notifications pour rappeler de trier correctement

Possibilité de visualiser l’historique des objets scannés

Flux utilisateur

Ouvre l’app → page Accueil

Prend une photo → app envoie l’image au backend / modèle IA

Affiche le Résultat → utilisateur peut corriger si besoin

Correction envoyée → IA apprend

Tableau de bord mis à jour → suivi de l’impact écologique

💡 Conseil pour Fagma :

L’interface doit être simple, intuitive et rapide, avec couleurs et icônes écologiques.

Les écrans doivent rester minimalistes, mais informatifs, pour ne pas surcharger l’utilisateur
